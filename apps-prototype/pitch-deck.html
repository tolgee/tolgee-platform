The idea

A Tolgee App that makes Git a first-class citizen

Connect

Point it at a repo

URL + branch + format (JSON, XLIFF, .properties, Android XML, YAML, PO…) + path pattern like locales/{languageTag}.json.

Push / Pull

One-click sync, with a diff first

See pending changes (+keys, ~translations, −keys) before anything moves. Commit directly or open a PR.

Auto-sync

Webhook-driven, hands-off

Translation changed in Tolgee → pushed to Git. New commit on the branch → pulled into Tolgee.

Conflicts

Pick a policy

Tolgee wins · Git wins · newest wins · always ask. Full activity log of every sync.
