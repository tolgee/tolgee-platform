# languages / FlagImage

FlagImage is a static element that is used to display country flag image based
on a country flag emoji. This way the flag image is consistent with our design
on all platforms.

## Flag List

<Unstyled
  style={{
    display: 'grid',
    gridTemplateColumns: 'repeat(auto-fill, minmax(120px, 1fr))',
    gap: '10px',
  }}
>
  {flagNames.map((flagName) => {
    const flagEmoji = String.fromCodePoint(
      ...flagName.split('-').map((hex) => parseInt(hex, 16)),
    );
    return (
      <div key={flagName}>
        <FlagImage flagEmoji={flagEmoji} width="64" />
        <div>{flagEmoji}</div>
        <div>{flagName}</div>
      </div>
    );
  })}
</Unstyled>
