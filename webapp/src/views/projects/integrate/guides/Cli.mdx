## Load translations with Tolgee CLI

To install Tolgee CLI globally run:

```sh
npm install --global @tolgee/cli
```

Login with project API key:

```sh
tolgee login "{{{apiKey}}}"
```

Pull translations into desired folder:

```sh
tolgee pull --project-id={{{projectId}}} --path ./i18n
```
